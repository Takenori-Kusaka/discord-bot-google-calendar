# 家庭内執事プロジェクト

## プロジェクト概要

木津川市在住の家族が地域で快適に暮らすための「家庭内執事」AIアシスタント。
Discord経由で地域情報、予定管理、生活サポートを提供する。

## 背景と目的

- 地域のイベント情報をDiscordに通知
- Googleカレンダーの予定を毎朝通知し、忘れ防止
- 超一流執事としての素養を備え、家族の生活を総合的にサポート
